# حديقتي 🌿 — تطبيق حديقة السطح

## كيف تبني الـ APK

### الخطوة 1: ارفع المشروع على GitHub
1. اصنع Repo جديد على github.com
2. ارفع محتوى هذا الـ ZIP

### الخطوة 2: انتظر GitHub Actions
- بعد الرفع مباشرة بيبدأ البناء تلقائياً
- بيخلص بـ 3-5 دقايق

### الخطوة 3: حمّل الـ APK
- روح على Actions → آخر run → Artifacts
- حمّل **حديقتي-debug.apk**
- ثبّته على موبايلك (فعّل "Unknown sources")

## ميزات التطبيق
- 📅 جدول الري والتسميد اليومي
- 📷 تعريف النباتات بالكاميرا (Plant.id API)
- 🌿 قائمة الـ 32 نبات كاملة
- ⚙️ إعدادات متزامنة مع Google Calendar
